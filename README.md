# Apache CouchDB (apache-couchdb)
Apache CouchDB is an open-source document-oriented NoSQL database that uses JSON for data storage, JavaScript for MapReduce queries, and a RESTful HTTP/JSON API for all database operations. It provides built-in replication and offline-first capabilities.
